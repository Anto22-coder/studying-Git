С помощью Git программисты и разработчики ориенируются ориентируется в коде и отслеживают измененияю Git помогает вернуть файлы в исходное состояние и видеть изменения, внесенные в определенный момент. Разработчик выполняет разные команды (например: commit, push), а все изменения синхранизируются с центральным репозиторием. Git - это систеа контроля версий, а GitHub - онлайн сервис, по сути социальная сеть. Одна из основных версий GitHub - быть единым местом для проектов с исходным кодом. Предпологается, что пользователь делится чем то полезным, а другие люди смогут участвовать в разработкею
